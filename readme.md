# Sheet Metal Equipment Mounting Bracket

**Precision Sheet Metal Design with Multi-Bend Forming & Manufacturing Features**

## Overview

This project is a sheet metal equipment mounting bracket designed in Autodesk Inventor as part of my sheet metal design practice. The model demonstrates industry-standard sheet metal techniques, including multiple bends, mounting features, cut-outs, reliefs, and manufacturable geometry.

The objective of this project was to improve my understanding of sheet metal workflows while creating a realistic production-ready component that could be manufactured using laser cutting and press brake operations.

---

## Features

- Multi-bend sheet metal construction
- Parametric design
- Mounting holes for fasteners
- Formed support tabs
- Bend reliefs
- Internal cut-outs to reduce weight and allow component clearance
- Rounded corners and fillets for improved manufacturability
- Symmetrical mounting geometry
- Flat pattern generation for manufacturing

---

## Design Process

The bracket was created using Autodesk Inventor's Sheet Metal environment by following a typical manufacturing workflow:

1. Create the base face
2. Add edge flanges
3. Form upper retaining brackets
4. Create mounting holes
5. Add clearance cut-outs
6. Apply bend reliefs
7. Generate the flat pattern for fabrication
